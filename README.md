# Nest JS + TypeORM

## Description

This is POC using a node backend framework called Nest JS and an ORM called TypeORM

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## Commands

```bash
# Create a ressource called article with all the config files.
$ nest g resource article
```
